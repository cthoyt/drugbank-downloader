Installation
============
The most recent release can be installed from
`PyPI <https://pypi.org/project/drugbank_downloader>`_ with:

.. code-block:: shell

    pip install drugbank_downloader

The most recent code and data can be installed directly from GitHub with:

.. code-block:: shell

    pip install git+https://github.com/cthoyt/drugbank-downloader.git

To install in development mode, use the following:

.. code-block:: shell

    git clone git+https://github.com/cthoyt/drugbank-downloader.git
    cd drugbank-downloader
    pip install -e .
